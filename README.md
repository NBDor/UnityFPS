# Unity-FPS
3D Arena shooter with both friendly and hostile AI

## Controls:

* W A S D - movement
* Q - throw a grenade.
* Left Mouse Click/Primary Mouse Button - Shoot your current gun (2 types)
* E - pickup a gun/grenade.
